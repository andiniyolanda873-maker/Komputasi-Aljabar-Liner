# Visualisasi Transformasi Cermin terhadap Sumbu-X

## Deskripsi

Program ini memvisualisasikan transformasi pencerminan terhadap sumbu-X menggunakan Python dan Matplotlib.

Rumus transformasi:

\[
(x, y) \rightarrow (x, -y)
\]

---

## Hasil Animasi

![Animasi Cermin](cermin_sumbu_x.gif)

---

## Warna Visualisasi

- Biru → Objek Asli
- Merah → Bayangan Cermin
- Ungu → Garis Cermin

---

## Cara Menjalankan

Install library:

```bash
pip install matplotlib numpy pillow
```

Jalankan program:

```bash
python nama_file.py
```

---

## Kesimpulan

Pencerminan terhadap sumbu-X mengubah nilai koordinat y menjadi negatif tanpa mengubah nilai x sehingga objek dan bayangan bersifat simetris terhadap sumbu-X.
